# 🛣️ AI-Based Pothole Detection using YOLOv8

This project detects potholes from images and videos using YOLOv8 and provides road quality analysis.

## 🚀 Features
- Detect potholes using AI
- Works on images and videos
- Calculates pothole count
- Displays Road Quality Index (RQI)
- Suggests safe vehicle speed

## 🧠 Technologies Used
- Python
- YOLOv8
- OpenCV
- Tkinter

## ⚠️ Note
Custom trained model and dataset are not included due to size limitations.
